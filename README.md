- http://cis.gov.pl/;
- https://github.com/cis-ncbj;

Centralny Ośrodek Informatyki (Center for Information Technology):
- http://www.coi.gov.pl/;
- https://github.com/coi-gov-pl;

- [OpenSourceDay 2016 presentation][2] by Krzysztof Suszyński.
